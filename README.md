# 小说推文文案生成器 🔥

> 基于爆款案例分析的AI智能文案生成系统，集成DeepSeek AI技术，专为小说推文创作者打造

## ✨ 项目特色

- 🎯 **6种文案风格**：工具分享型、避雷警告型、成果展示型、工具合集型、避坑指南型、干货合集型
- 🤖 **AI驱动创作**：集成DeepSeek AI，确保每次生成独特原创内容
- 🛠️ **必含魔音工坊**：每篇文案都包含魔音工坊AI配音工具推荐
- 📊 **智能工具组合**：基于真实爆款案例，随机组合推荐实用工具
- 🎨 **现代化界面**：响应式设计，支持移动端访问
- ⚡ **实时生成**：快速响应，一键生成完整文案套装

## 🚀 功能亮点

### 📝 文案生成
- **双文案输出**：一次生成2篇不同风格的专业文案
- **智能字数控制**：根据工具数量自动调整详细程度
- **专业术语融入**：深度结合小说推文领域知识

### 🏷️ 标题生成
- **5个热门标题**：基于爆款标题模板智能生成
- **数据驱动**：参考真实播放量、粉丝数等数据
- **风格匹配**：标题与文案风格完美契合

### 🖼️ 配图建议
- **2个配图方案**：提供具体的视觉设计建议
- **场景化描述**：详细的素材组合和效果说明
- **平台适配**：考虑不同平台的展示特点

## 📋 工具数据库

## 🎯 使用指南

### 快速开始
1. **选择文案风格**：为两篇文案分别选择不同风格
2. **设置工具数量**：6-15个工具，影响详细程度
3. **一键生成**：点击生成按钮，AI自动创作
4. **获取结果**：标题、文案、配图建议一应俱全

### 风格说明
| 风格类型 | 特点描述 | 适用场景 |
|---------|---------|----------|
| 工具分享型 | 友好专业，乐于分享 | 建立专业形象，分享实用经验 |
| 避雷警告型 | 紧迫感，过来人口吻 | 通过踩坑经历建立信任 |
| 成果展示型 | 自信成功，激励他人 | 用数据结果吸引关注 |
| 工具合集型 | 系统全面，实用导向 | 提供完整解决方案 |
| 避坑指南型 | 理性分析，客观中立 | 预警式内容，增加价值感 |
| 干货合集型 | 干货满满，超值分享 | 全面知识整理分享 |

### 操作技巧
- **快捷键**：`Ctrl/Cmd + Enter` 快速生成，`Ctrl/Cmd + C` 复制全部
- **工具数量**：≤10个详细介绍，>10个分类罗列
- **批量操作**：支持一键复制全部内容和导出文档
- **响应式**：支持手机、平板、电脑多端访问

## 🚀 部署指南

### 本地开发
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动应用
export PORT=8002
python app.py

# 3. 访问应用
open http://localhost:8002
```

### 生产部署
```bash
# 使用Gunicorn部署
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:8002 app:app

# 或使用uWSGI
pip install uwsgi
uwsgi --http :8002 --module app:app --processes 4 --threads 2
```

### Docker部署
```dockerfile
FROM python:3.10-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
EXPOSE 8002
CMD ["python", "app.py"]
```
